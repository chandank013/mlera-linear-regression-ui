# MLera Linear Regression UI

## 📌 Overview

This project is a frontend recreation of **MLera's Linear Regression learning module UI**, built as part of the **MLera Internship Assignment – Machine Learning for Education**.

The goal of this assignment is to accurately recreate the UI shown in the provided screen recording using **Next.js (App Router)** and **Tailwind CSS**, while following a clean, reusable, and component-based architecture.

---

## 🚀 Tech Stack

* **Next.js** (App Router)
* **React.js**
* **Tailwind CSS**
* **TypeScript**
* **Vercel** (Deployment)

---

## ✨ Features

* Pixel-accurate UI recreation based on the screen recording
* Component-based and reusable architecture
* Fully responsive layout
* 🌗 **Dark / Light mode toggle** with theme persistence
* Clean folder structure and readable code
* Deployed and hosted on Vercel

---

## 🎨 Creative Enhancement

A subtle UI enhancement was added in the form of:

* Improved card hover effects and transitions
* Slightly refined typography and spacing for better readability

These improvements maintain simplicity while enhancing user experience.

## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/chandank013/mlera-linear-regression-ui.git

# Navigate to project directory
cd mlera-linear-regression-ui

# Install dependencies
npm install

# Run development server
npm run dev
```

Open `http://localhost:3000` in your browser.
