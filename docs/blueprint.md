# **App Name**: MLera UI Recreator

## Core Features:

- UI Recreation: Recreate the UI of the Linear Regression learning module as demonstrated in the provided video link. Match the design closely, focusing on accuracy.
- Component-Based Architecture: Implement the UI using reusable components for elements like the Navbar, content areas, cards, and footer.
- Responsive Design: Ensure the recreated UI is fully responsive and adapts seamlessly to different screen sizes.
- Dark/Light Mode Toggle: Implement a toggle in the navbar to switch between dark and light themes. Ensure smooth theme switching with state persistence.
- Enhanced Layout: Improve the layout of a section, or design a custom card with enhanced visuals. The goal is meaningful and simple enhancements to make design more visually pleasant.
- Vercel Deployment: Deploy the application on Vercel to meet deployment requirements, ensuring the live link is fully functional and submitted along with GitHub link.

## Style Guidelines:

- Primary color: Deep blue (#1A237E), inspired by the 'Machine Learning for Education' internship to create a professional, trustworthy learning atmosphere.
- Background color: Very light gray (#F5F5F5), creating a clean, distraction-free learning environment.
- Accent color: Soft violet (#9FA8DA) to provide a gentle contrast and highlight key interactive elements, analogous to the primary color for visual coherence.
- Body and headline font: 'Inter', a sans-serif font known for its clean and modern appearance, for the app's UI to be both easily read and accessible.
- Use minimalist icons for navigation and interactive elements to keep a distraction-free and simple visual profile.
- Maintain a clean and intuitive layout, optimizing the spacing and alignment for ease of navigation, information consumption, and ease of understanding. Focus on visual rhythm and logical structure.
- Implement subtle transitions for theme switching and hover effects, enhancing the overall experience with gentle and clear interactive feedback.
